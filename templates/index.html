This is an art gallery management system implemented in python and PostgreSQL database following the Web Front-End Option.

The application can be used to make people aware of the information related to upcoming exhibition organized by galleries. Customers can contact artists if they want to buy their art pieces. Customers can also contact the gallery if they have any inquiries and suggestions.
